API Access

Public API Endpoints

This project features a complete RESTful API that powers the entire frontend. The following public endpoints are available for use:

  • GET /api/torrents - Fetch all torrents.
  • GET /api/torrents?q=[keyword] - Search for torrents by name.
  • GET /api/torrents?categories=[cat1,cat2] - Filter torrents by category.
  • GET /api/torrents/recent - Get the 50 most recently uploaded torrents.
  • GET /api/torrents/top?category=[cat] - Get the top 100 torrents, optionally filtered by category.
  • GET /api/torrents/:id - Get details for a single torrent.
  • GET /api/users/profile/:username - Get a user's public profile and stats.

Protected endpoints for uploading, posting, and following require a valid JWT Bearer Token obtained via login or registration.
